Lets create a program using ruby classes!

Hi students! today we will be going over the creation of a program using ruby classes!

Ruby, in all of its greatness, can allow you to create any kind of object you want! In our example today we will be creating a class called "Computer", which will generate instances of that class which can then act to manipulate its own imaginary files.

Firstly, we should create a class called "Computer", and give it an initialize method that takes no parameters. The body of initialize itself, should be empty. It should look something like this:

class Computer
def initialize
end
end

Next, we should be passing our initialize method some parameters and setting up a few instance variables. Here we update our initialize method to take 2 parameters, username and password. In initialize, set the instance variables to @username and @password.In our initialize method we also want to add another instance variable, @files, which is equal to an empty hash. This will act to keep track of our files later on. The code should look like this:

class Computer
def initialize (username, password)
@username = username
@password = password
@files = {}
end
end

Now, let's introduce some class variables into the mix! We will be using @@users to keep track of the users on our computer. Add a new class variable to our computer class called @@users, it should equal an empty hash. Next, in the initialize method, set @@users[username] equal to password, so that our @@users hash keeps usernames as keys with each username's password as the associated value. Your code should look like this:

class Computer
@@users = {}
def initialize (username, password)
@username = username
@password = password
@files = {}
@@users[username] = password
end
end

Now let's add another method, create. Inside our Computer class, define a method called create, with a single parameter, filename. Inside create, declare a variable called time and set it equal to the current time, this is done by using Time.now. Inside create, add a new key:value pair to @files hash. Use the filename key to store the value time. Finally, use puts to return a message telling the user that a new file was created. Your code should look something like this:

class Computer
@@users = {}
def initialize (username, password)
@username = username
@password = password
@files = {}
@@users[username] = password
end
def create (filename)
time = Time.now
@files[filename] = time
puts "#{filename} was created by #{@username} at #{time}"
end
end

Almost done guys! We need to now find a way to get to our @@users class variable! We will need to create a method to reach it. because @@users is a class variable, we can use a class method to get it. Most of the methods used to far have been instance variables, which work on specific instances/objects. A class method belongs to the class itself. Add a new class method to your Computer class called Computer.get_users, taking no parameters. This new method should return the @@users class variable and puts a message!. Your code should look like this:

class Computer
@@users = {}
def initialize (username, password)
@username = username
@password = password
@files = {}
@@users[username] = password
end
def create (filename)
time = Time.now
@files[filename] = time
puts "new file has been created!"
end
def Computer.get_users
@@users
puts "Let's get those users for you!"
end
end
