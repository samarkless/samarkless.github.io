Enumerable#map

What does this method do?

What are enumberables? Basically these are a set of methods which traverse, search, sort and manipulate the user's input. What is the Map Method? The Map method basically loops through what the user has inputed, say an array, and then preforms a function on the elements of said array.

Consider the example below taken from ruby.bastardsbook.com:

arr = [1,2,3,4,5]

brr = arr.map{|x| -x}

puts arr.join(", ") #=> 1, 2, 3, 4, 5

puts brr.join(", ") #=> -1, -2, -3, -4, -5

In the example above, the map method returns a copied version of the elements that it is called upon, the map method takes an enumerable object and a block, and runs the block for each element, it then gives an output which is each returned value from the block.
